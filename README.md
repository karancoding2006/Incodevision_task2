# Responsive Landing Page

## Project Overview
This project is a simple, modern, and responsive landing page developed using only HTML5 and CSS3. It includes a navigation bar, a hero section with a call-to-action button, and three feature cards. The layout is designed to work smoothly on both desktop and mobile devices.

## Features
- Responsive Navigation Bar
- Hero Section with Heading, Description, and Button
- Three Feature Cards
- Clean and Modern User Interface
- Mobile-Friendly Design
- Built Using Only HTML and CSS

## Technologies Used
- HTML5
- CSS3

## Project Structure

## File Description
modern_index.html – Contains the structure of the webpage.
